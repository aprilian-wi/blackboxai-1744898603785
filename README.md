# MeningitisGo - Vaksin Meningitis Home Service

## Project Overview
MeningitisGo is a web application that provides a convenient home service for Meningitis vaccinations. This service is designed for individuals, families, and groups, offering vaccination and official ICV (International Certificate of Vaccination) issuance for Hajj and Umrah pilgrims without the hassle of queuing.

## Installation
To set up this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd meningitisgo
   ```

3. **Open your `index.html` file in a web browser:**
   You can do this by double-clicking on the `index.html` file or dragging it into your browser.

> Note: The project uses external resources for styling and fonts; ensure you have an internet connection to load Tailwind CSS and Font Awesome.

## Usage
Simply open the `index.html` file in your browser to use the application. The interface provides information about the vaccination service, the medical team, service areas, and costs associated with the Meningitis vaccination.

- Users can click on the **"Pesan Sekarang via WhatsApp"** button to directly contact the service provider and schedule a vaccination appointment.

## Features
- **Home service for Meningitis vaccination**: Vaccinations are conducted at the customer's location.
- **Professional medical team**: Experienced and certified healthcare professionals administer the vaccinations.
- **Standardized procedures**: All services comply with WHO and Indonesian Ministry of Health standards.
- **Easy scheduling through WhatsApp**: Customers can easily reach out and set a convenient appointment.
- **Informative sections**: The site provides detailed information about service areas, costs, and customer testimonials.

## Dependencies
This project uses the following external libraries:
- **Tailwind CSS**: For styling, loaded from a CDN.
- **Font Awesome**: For icons, loaded from a CDN.
- **Google Fonts**: The Inter font is used for typography.

No additional JavaScript packages are used in this project, so a `package.json` file is not necessary.

## Project Structure
The project consists of a single HTML file with embedded CSS styling. The structure is as follows:

```
/meningitisgo
├── index.html
```

The `index.html` file contains:
- Hero section with service introduction.
- Information about the medical team and certifications.
- Service area details.
- Pricing information and customer testimonials.
- Contact details and operational hours.

## Conclusion
MeningitisGo aims to simplify the process of getting vaccinated against Meningitis, ensuring comfort, efficiency, and safety for its users. For further information or collaboration inquiries, feel free to contact us via WhatsApp.
